Chainlink

About

ChainLink relies on a set of trusted parties to provide reference asset prices to the Ethereum blockchain. ChainLink also offers a service for users to request off-chain data from trusted oracle providers. ChainLink provides free price reference data feeds for the ETH/USD and BTC/USD pairs which rely on 14 trusted parties to submit reference data every 20 minutes. Unlike MakerDAO, however, ChainLink does not have a governance process to decide the participating parties. ChainLink itself is responsible for adding/removing parties from the system, centralizing control of the reference oracle.

ChainLink allows application developers to run their own ChainLink node or request other off-chain data, but requires payment in LINK. The LINK token is an ERC-223 token that implements a custom function that allows for the transfer of data along with token transfers. This mechanism is used to query and transfer resulting data from ChainLink nodes to on-chain smart contracts. If application developers wish to query non-reference data, they can either run their own ChainLink node, or query one of ChainLinkâ€™s trusted nodes. LINK is charged by each node that is queried. Users can call multiple nodes to achieve the wanted degree of decentralization, but the user would be charged for every query.

ChainLink gives developers the option to run their own ChainLink node that responds to their own requests, but developers still must buy and send the LINK token to themselves every request to respond successfully. ChainLink will also be establishing a whitelist for all on-chain reference data (currently free). The ChainLink company will control additions to this list, further centralizing their control over the service.
